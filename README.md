# Audit Intelligence System

## 📌 Overview
The Audit Intelligence System is an AI-powered solution designed to automate bank statement analysis. It classifies and labels transactions based on their descriptions using intelligent processing techniques.

---

## 🚀 Objective
To reduce manual effort in auditing by automatically identifying transaction types such as salary, ATM withdrawal, shopping, etc., from raw bank statement data.

---

## ⚙️ Features
- Automatic transaction labeling
- Intelligent text-based classification
- Supports bank statement CSV input
- Clean and structured output
- Scalable for AI-based enhancements (RAG, LLMs)

---

## 🧠 Approach
1. Load bank transaction data from CSV  
2. Preprocess transaction descriptions  
3. Apply classification logic (rule-based / AI-based)  
4. Generate labeled output  

---

## 📂 Project File
- `audit_intelligence.ipynb` → Contains full code, explanation, and results

---

## ▶️ How to Run
1. Open the notebook in Jupyter Notebook / Google Colab  
2. Run all cells step-by-step  
3. View labeled transaction output  

---

## 📊 Sample Output
| Transaction Description | Label |
|------------------------|-------|
